Installation guide

About 15 minutes, start to finish.

Before you start

You'll need two things:

A Google account

Your dashboard lives in Google Sheets — free to use, data stays in your account. If you don't have one, create a free account here.

macOS or Linux

The agent runs on macOS (Apple Silicon) and Linux (x86-64). Windows support is planned for a future release.

Part 1 — Create your self-hosted dashboard

This is a one-time setup. You'll create a private Google Sheet that acts as your family dashboard. Your data never leaves your Google account.

1 Create a new Google Sheet

Go to sheets.google.com and create a blank spreadsheet. Give it a name like Ludex.

2 Open the Apps Script editor

In the sheet, click Extensions → Apps Script. A code editor opens in a new tab.

3 Paste the Ludex script

Download the latest backend.zip from the releases page. Copy the contents of each .gs file into the Apps Script editor, creating a matching file for each one.

ℹ️ The script editor uses + to add new files. Match the filenames exactly (e.g. Code.gs, Menu.gs, …).
4 Set up your dashboard

Save the project (Ctrl+S / ⌘S), then go back to your Google Sheet and reload the page. A Ludex menu appears in the toolbar.

Click Ludex → ① Set credentials and enter:

  • A shared key — any long random string, e.g. my-secret-key-2024
  • An admin password — used for sensitive operations; keep it separate from the shared key

Click Save. Ludex creates all the data tabs automatically.

5 Deploy as a web app and get your Dashboard URL

Back in the Apps Script editor:

  1. Click Deploy → New deployment
  2. Select type: Web app
  3. Execute as: Me
  4. Who has access: Anyone
  5. Click Deploy and authorize when prompted

Copy the Web app URL — it ends in /exec. This is your Dashboard URL. Keep it along with your shared key; you'll need both when installing the agent.

⚠️ Don't paste the regular Google Sheet URL — that won't work. You need the /exec URL from the deployment dialog.
Screenshot: Apps Script deployment dialog — showing where to copy the /exec URL

Part 2 — Install the agent on your child's computer

Repeat this for each computer you want to monitor. The agent runs as the logged-in user — no admin rights required.

1 Download the agent

On the child's computer, download the latest release for the right platform from the releases page:

  • macOS — ludex-macos-arm64.tar.gz
  • Linux — ludex-linux-x86_64.tar.gz

Extract the archive. You'll find a single ludex binary inside.

2 Run the installer

Double-click the ludex binary (or run it from a terminal). A browser window opens:

Screenshot: Ludex installer form in the browser

Enter:

  • Dashboard URL — the /exec URL from Part 1, step 5
  • Shared key — the key you set in Part 1, step 4

Click Install. Ludex validates the connection and registers the agent as a background service.

✅ Once installed, the agent starts automatically at login. You don't need to do anything else.
3 Add the child's friendly name

In your dashboard, click Ludex → Edit names. Each computer that has checked in will appear in the people tab. Replace the system username with your child's name — this is what appears in the dashboard.

Next steps

You're set up. A few things worth doing now:

  • Define activities — Ludex ships with common games and apps pre-defined. Review and adjust them for your family. See the activity guide.
  • Set limits — Click Ludex → Edit activity limits to set daily time limits and warning thresholds per activity.
  • Add an alert email — In the people tab, add your email address so you're notified when a daily limit is crossed.
ℹ️ The dashboard updates every few minutes as the agent syncs. Click Ludex → Refresh dashboard to force an immediate refresh.
